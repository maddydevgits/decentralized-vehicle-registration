# decentralized-vehicle-registration
Decentralized Vehicle Registration is a project on Blockchain and is a dapp which makes vehicles registration

# Softwares
1. Ganache
2. Nodejs
3. Truffle 
4. Python3
5. VS Code

# Packages

2. pip install flask
